<b>Mailfoundr</b> is a bulk email-sending application built using Python's Tkinter library. Developed as a final project for the <b>Data Science and AI workshop</b> at Nepal College of Information Technology (NCIT), this app allows users to efficiently send personalized emails to multiple recipients at once.
